# Release Notes - v1.0.0

## Summary

Initial public release of SmartFarmWiFiGuard for ESP32.

## Included in this release

- Wi-Fi connection management for ESP32
- Automatic reconnection logic
- Configurable reconnect interval
- Configurable reconnect attempt limit
- RSSI reporting
- Signal quality classification
- Event callbacks for connection lifecycle
- Optional automatic device restart after extended Wi-Fi outage
- Arduino example sketch
- Professional documentation for GitHub publication

## Intended audience

This release targets developers, researchers, and engineers building IoT and smart farming systems on ESP32 using the Arduino framework.

## Next planned improvements

- MQTT watchdog support
- Farm sensor integration examples
- Offline telemetry buffering
- Expanded deployment documentation
